La página entera se encuentra dentro de un módulo div.page. Creado para hacer páginas básicas con un contenedor div.page, un div.page__header y div.page__footer opcional.

Puede que estos componentes usen pequeños bloques de javascript para realizar algunas interacciones básicas con la UI. Sin embargo, pese a que la intención del proyecto es enfocarse en CSS, sientase libre de usar cualquier framework o librería que desee para realizar interacciones y animaciones más avanzadas.

Callouts

Los callouts son aquellas secciones de contenido "populares" que usualmente contienen un ícono o imagen, un título y una pequeña descripción para llamar la atención del usuario. Generalmente se encuentran dentro de una grid, como se muestra a continuación:

Thumbnails

Los thumbnails usualmente se encuentran dentro de una grid o galería, se usan para mostrar una vista de un contenido en específico. Comúnmente contienen una imagen encima del título y un poco más de texto debajo para reforzar la información que se presenta al usuario. El thumbnail entero es un ancla hacia el contenido escondido que se espera mostrar.

Thumbnails are usually found within a grid or gallery, as a teaser or preview of more content. The common use is to have an image on top of a title and text below, while the entire thumbnail is an anchor link.

Ejemplo de Ventana Modal

Este es un buen ejemplo de como se verían las ventanas modales en una aplicación sencilla, haciendo uso de la pantalla completa para llamar la atraer al usuario y captar su atención en el contenido que se le presenta.

Dado que la ventana modal se usa a modo de pantalla completa, hay que ser consciente de que su contenido debe ser realmente importante y dependiente de las acciones del usuario para que este no sienta que es algo invasivo.
